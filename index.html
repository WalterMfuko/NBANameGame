How to Play the NBA Name Game

Basic Rules:

1. Players take turns naming NBA players (past or present)

2. The first letter of your player's first name must match the first letter of the previous player's last name

Example:

- If the previous player was "Michael Jordan"

- The next player must start with "J" (like "Jason Kidd")

- Then the next player must start with "K" (like "Kevin Durant")

Invalid Moves:

  • Using a name that's already been used in this game
  • Using a name with the wrong starting letter
  • Using a name that's not in our NBA player database

Hints:

Each player gets 2 hints per turn. Click the hint button when needed.
