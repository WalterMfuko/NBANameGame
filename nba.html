How to Play: NBA Players

Basic Rules:

1. The computer and you take turns naming NBA players

2. The first letter of your player's first name must match the first letter of the previous player's last name

Example:

- If the computer plays "Michael Jordan"

- You must enter a player starting with "J" (like "Jason Kidd")

- Then the computer will play a name starting with "K"

Invalid Moves:

  • Using a name with the wrong starting letter
  • Using a name that's not in our NBA player database

Hints:

You get 2 hints per turn. Click the hint button when needed.
